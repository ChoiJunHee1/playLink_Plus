PlayMD 서비스 소개

● Cafe24쇼핑몰 과 PlayMD 상품 Mapping 으로 온라인 판매 상품 정보 수집 시스템

● 수집된 온라인 판매 상품 택배 송장번호 입력 및 발송 시스템

● 온라인쇼핑몰 상품 Mapping으로 상품재고 연동 시스템

PlayMD 이용방법

1. Cafe24 쇼핑몰 에서 PlayMD 앱 설치

2. PlayMD 앱 접속 후 Cafe24 API 등록

3. Cafe24 API 등록 후 온라인판매 상품 수집, 송장발송, 재고연동 서비스 이용
